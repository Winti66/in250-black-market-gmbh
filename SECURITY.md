# Das sind die Sicherheitsrichtlinien von Black Market Software Gmbh

## Unterstützte Versionen
Wir schützen derzeit nur die aktuellste Version unserer Code-Festung.

| Version | Unterstützt     |
| ------- | ---------------- |
| 1.0.x   | ✅ Ja            |
| < 1.0   | ❌ Nein          |

## Ein Sicherheitsproblem melden
Wenn du eine dunkle Schwachstelle oder einen verfluchten Bug entdeckst, nutze bitte **nicht** die öffentlichen Issues. 

1. Sende eine verschlüsselte Botschaft an: **security@blackmarket-software.com**
2. Beschreibe die Bedrohung so genau wie möglich.
3. Unsere Magier werden die Lücke innerhalb von 48 Stunden prüfen.

Vielen Dank, dass du hilfst, das Königreich zu schützen!
